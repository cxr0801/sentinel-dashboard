## 🔄 自動化流程（已上線）

```
每日 Apify 排程爬蟲執行
       ↓ Run succeeded
Apify HTTP Webhook 觸發
       ↓ POST → api.github.com/repos/cxr0801/sentinel-dashboard/dispatches
GitHub Actions 自動執行（約 25 秒）
  1. 安裝 Python 依賴
  2. 執行 auto_daily_analysis.py（Token 從 GitHub Secrets 注入）
  3. git commit & push 更新後的 sentinel_dashboard.html
       ↓
GitHub Pages 自動重新部署（約 36 秒）
       ↓
https://cxr0801.github.io/sentinel-dashboard/sentinel_dashboard.html
       ↑ 管理層每天打開即是最新資料
```

## 🔑 重要設定

| 項目 | 值 |
|------|-----|

| GitHub Pages URL | `https://cxr0801.github.io/sentinel-dashboard/sentinel_dashboard.html` |
| GitHub Secret | `APIFY_API_TOKEN`（在 Repo Settings > Secrets）|

| Apify Webhook URL | `https://api.github.com/repos/cxr0801/sentinel-dashboard/dispatches` |
| Apify Webhook Event | `Run succeeded` |
| Apify Webhook Payload | `{"event_type": "apify-done"}` |
