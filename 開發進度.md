## 📋 本機執行方式

```bash
# 在專案目錄下執行
python auto_daily_analysis.py
```

執行後依序完成：
1. 查詢最新 Apify 執行紀錄
2. 下載資料集
3. 情緒分析
4. 產出 Excel 報告
5. 更新儀錶板 HTML

```bash
# 本機預覽儀錶板
python -m http.server 8899
# 開啟 http://localhost:8899/sentinel_dashboard.html
```
