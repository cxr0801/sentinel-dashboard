# 每日自動分析系統 - 設定指南

## 📋 前置準備

### 1. 安裝 Python 套件
```bash
pip install requests pandas openpyxl
```

### 2. 取得 Apify API Token
1. 登入 https://apify.com
2. 點擊右上角頭像 → **Settings**
3. 左側選單 → **Integrations**
4. 找到 **Personal API tokens** 區塊
5. 點擊 **+ Create new token**
6. 輸入名稱（例如：daily-scraper），點擊 **Create**
7. **複製 Token**（只會顯示一次！）

範例：`apify_api_1a2B3c4D5e6F7g8H9i0J1k2L3m4N5o6P7q8R`

### 3. 修改腳本設定
打開 `auto_daily_analysis.py`，修改第 16 行：
```python
APIFY_API_TOKEN = "apify_api_1a2B3c4D5e6F7g8H9i0J1k2L3m4N5o6P7q8R"  # ← 貼上你的 Token
```

## ⏰ 設定每日自動執行

### 🪟 Windows 設定（工作排程器）

#### 方法一：圖形界面

1. **開啟工作排程器**
   - 按 `Win + R`
   - 輸入 `taskschd.msc`
   - 按 Enter

2. **建立基本工作**
   - 右側點擊 **建立基本工作**
   - 名稱：`靠北長官每日分析`
   - 描述：`每天早上8點自動下載與分析資料`
   - 點擊 **下一步**

3. **觸發程序**
   - 選擇 **每天**
   - 設定時間：`08:00:00`（建議比 Apify 執行時間晚 1 小時）
   - 點擊 **下一步**

4. **動作**
   - 選擇 **啟動程式**
   - 程式或指令碼：
     ```
     C:\Users\你的使用者名稱\AppData\Local\Programs\Python\Python311\python.exe
     ```
     （路徑可能不同，輸入 `where python` 查詢）
   
   - 新增引數：
     ```
     auto_daily_analysis.py
     ```
   
   - 開始於（資料夾路徑）：
     ```
     D:\靠北長官分析\
     ```
     （填入你放腳本的資料夾）

5. **完成**
   - 勾選 **當按一下完成時，開啟此工作的內容對話方塊**
   - 點擊 **完成**

6. **進階設定**
   - **一般** 頁籤 → 勾選 **以最高權限執行**
   - **設定** 頁籤 → 取消勾選 **如果工作執行時間超過下列時間，停止工作**
   - 點擊 **確定**

#### 方法二：命令列（快速）

```powershell
# 開啟 PowerShell（系統管理員）
schtasks /create /tn "靠北長官每日分析" /tr "python D:\靠北長官分析\auto_daily_analysis.py" /sc daily /st 08:00 /rl highest
```

### 🍎 macOS / Linux 設定（cron）

1. **編輯 crontab**
   ```bash
   crontab -e
   ```

2. **加入排程**（按 `i` 進入編輯模式）
   ```bash
   # 每天早上 8:00 執行
   0 8 * * * cd /Users/你的使用者名稱/靠北長官分析 && /usr/bin/python3 auto_daily_analysis.py >> daily_log.txt 2>&1
   ```

3. **儲存並離開**
   - 按 `Esc`
   - 輸入 `:wq`
   - 按 Enter

4. **確認排程已加入**
   ```bash
   crontab -l
   ```

## ⚠️ 常見問題

### Q1: 執行失敗，顯示 "找不到最新執行紀錄"
**原因**：Apify 沒有成功執行，或 API Token 錯誤
**解決**：
1. 檢查 Apify 排程是否正常執行
2. 確認 API Token 正確

### Q2: 執行失敗，顯示 "ModuleNotFoundError: No module named 'requests'"
**原因**：缺少 Python 套件
**解決**：
```bash
pip install requests pandas openpyxl
```

### Q3: Windows 工作排程器顯示成功，但資料沒更新
**原因**：Python 路徑錯誤
**解決**：
1. 打開 PowerShell
2. 輸入 `where python`
3. 複製完整路徑到工作排程器

### Q4: 想改成每天下午 3 點執行
**Windows**：工作排程器 → 編輯觸發程序 → 改時間
**macOS/Linux**：`crontab -e` → 改成 `0 15 * * *`

## 📧 進階：執行完自動發 Email 通知

在 `auto_daily_analysis.py` 最後加上：

```python
def send_email_notification(stats):
    import smtplib
    from email.mime.text import MIMEText
    
    # Gmail 設定（需開啟應用程式密碼）
    sender = "your_email@gmail.com"
    password = "your_app_password"
    receiver = "your_email@gmail.com"
    
    msg = MIMEText(f"""
    今日輿情報告已完成
    
    總貼文數：{stats['total']}
    負面聲量：{stats['negative']} ({stats['negative']/stats['total']*100:.1f}%)
    平均按讚：{stats['avgLikes']}
    
    儀錶板已更新，請查看。
    """)
    
    msg['Subject'] = f"靠北長官輿情報告 {datetime.now().strftime('%Y-%m-%d')}"
    msg['From'] = sender
    msg['To'] = receiver
    
    with smtplib.SMTP_SSL('smtp.gmail.com', 465) as smtp:
        smtp.login(sender, password)
        smtp.send_message(msg)

# 在 main() 最後加上
send_email_notification(stats)
```
